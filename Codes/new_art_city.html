Project Overview: Interactive 3D Environment

This project focuses on **virtual space design** within the New Art City platform, leveraging custom-modeled assets from Blender to create an immersive, aesthetically pleasing suite for users to experience.

Process and Design Description

For my virtual space, I decided to create an aesthetic room that almost looks like a **suite** with various **polygons on the ground** and the **sky blue feature** in the background.

The inside part of the box was designed like a suite, containing **plants** and a **couch** so that people can relax. I adjusted the exposure of the ceiling lights so that the suite could gain more lighting.

I created the main object using **Blender** while referencing YouTube tutorials, and then exported the file to add the custom polygons and other objects to my world. I also created the **walls and floors with different types of styling and textures** so that the whole room could be visually appealing.
