Project Overview: Data Bending with TextEdit

This project is the first in an experimental digital art series, focusing on **glitch art** created by manually manipulating a file's code. This technique is often referred to as **data bending**.

Category:

Data Manipulation / Glitch Art

Tools Used:

**TextEdit** (Mac), Preview

File Source:

JPEG Image

Artwork

Process Description

I used **TextEdit** on Mac to open the original image that was in JPEG format. Then I **copied and pasted a few sections** after approximately **20% into the file** (to avoid corrupting the file header). Finally, I saved the file and opened that image in **"Preview"** mode to help see the glitches of what changed according to my image.
