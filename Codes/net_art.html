Project Screenshot

Detailed Description and Intent

I decided to create a **10-page website** that contains 10 pages of artwork and presentations that I've been working on for the past years of college. They all contain a lot of **videos and images** I imported using the GitHub code structure, and the overall experience reminded myself of creating a **portfolio review** covering all the work I've done so far.

This review also helped me to demonstrate my passion for creating projects so I could hone my skills using **graphic design and code**. All of the headers, which link to the reviews, start on the beginning of the first page. To go back from any page, the user simply presses the back button on the browser (Chrome, etc.).
