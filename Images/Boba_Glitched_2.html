Project Overview: Data Bending with Audacity

This second piece in the series explores a more complex data bending technique using **Audacity**, a sound editor, to manipulate the visual data of an image file. This process is a controlled corruption of the image to produce digital artifacts.

Category:

Audio Software Data Manipulation

Tools Used:

**Audacity**, Preview

File Format:

TIF (Tagged Image File Format)

Artwork

Process Description: Boba Glitched 2 (Audacity)

I first used **Audacity** software to import my image as a **.tif** file. Next, I opened the raw form and **copied a section** of the data. Then I used the **"reverse" effect** on selected sections to enhance the **waves pattern** of the resulting distortion. Lastly, when I saved my image, I exported the raw image and opened it in **"Preview"** mode to see the glitches of what changed according to my image.
